# Note App
## First release
https://your-note-mw51.onrender.com/

## Installation 
> Note requires Python 3.10+
```sh
git clone https://github.com/MykolaKrylevych/note.git
```
```sh
pip install -r requirements.txt
```

## Important 
Do not forget about set the next data in .env file:
```sh
FLASK_SECRET_KEY=
DATABASE_URI=
MAIL_SERVER=
MAIL_USERNAME=
EMAIL_PASS=
MAIL_PORT=
MAIL_USE_SSL=
```
